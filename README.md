# Card-game
Карточная игра "Пьяница" для двух игроков
Правила игры:
- Каждый игрок получает по 5 уникальных карт;
-Карты открываются сверху колоды;
-Тот, чья карта оказывается старше, 
 забирает обе вскрытые карты и кладет их 
 под низ своей колоды (сначала карту первого игрока,затем второго игрока);
-Большая карта бьет меньшую. Исключение: 0 побеждает 9;
-Проигрывает тот, кто остается без карт;
-Если за 1.000.000 ходов победитель не найден, объявляется "botva".
